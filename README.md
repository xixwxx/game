# One Room Roguelike

단일 화면 아레나에서 몰려오는 적을 막아내는 웨이브 기반 로그라이크.
순수 HTML5 Canvas + JavaScript (프레임워크 없음).

## 조작

- **이동**: WASD
- **공격/조준**: 방향키 (누른 방향으로 발사)

## 시스템

- **레벨업 보상** — 적을 잡아 XP를 모으면 보상 카드 3장 중 하나를 선택 (tier: bronze 등).
- **상점** — 골드로 하위/최종 아이템 구매. 같은 타입을 모으면 **세트 효과** 발동.
- **캐릭터 해금** — 희귀 재화 **Shards**로 메인 화면에서 캐릭터 구매 (기본형 / 화력형 / 쌍발형).
- **보스** — 5웨이브마다 중간 보스, 10웨이브마다 진짜 보스.
- **융합탄** — 탄을 조합해 상위 탄 제작 (게임 내 "융합표" 버튼에서 조합표 확인).

## 실행

정적 파일이라 빌드가 필요 없다.

```powershell
# 로컬 서버로 열기
python -m http.server 8000
# → 브라우저에서 http://localhost:8000

# 또는 index.html 을 브라우저로 직접 열기
```

## 구성

- `index.html` — HUD·오버레이·캔버스 마크업
- `game.js` — 게임 로직 전체 (상태머신·웨이브·전투·보상·상점·융합)
- `styles.css` — UI 스타일
