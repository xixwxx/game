## 1. 핵심 목표

**웨이브 30~70에서도 성장의 재미와 새로운 느낌이 지속되어야 한다.**

단순 수치 증가(+데미지, +체력의 반복)는 성장감이 아니다. 후반에도 빌드가
**질적으로** 변해야 한다 — 새 세트 효과가 발동하고, 융합이 바뀌고, 플레이
스타일이 달라지는 순간이 후반 구간에도 계속 나와야 한다.

### 1.1 주력 수단: 세트 간 시너지

빌드 분기 심화의 우선 방향은 **보상 타입(세트) 간 교차 시너지**다.

- 현재: 보상 타입 10종(화력/속도/생존/도탄/관통/행운/폭발/독/번개/빙결)이
  각자 N세트 효과를 갖지만, 타입끼리는 독립적이다.
- 지향: 타입 **조합**에 교차 효과를 부여한다 (예: 독+번개, 빙결+폭발).
  세트 두 개를 같이 키울 전략적 이유가 생기면, 같은 보상 풀에서도
  빌드 경로가 조합 수만큼 분기한다.
- 탄환 융합(21종+)은 이미 구축된 분기 축이므로 유지하되, 신규 작업의
  우선순위는 세트 교차에 둔다.

## 2. 고정 전제

**단일 룸 아레나 + 웨이브 루프 구조는 바꾸지 않는다.**
맵 확장, 메타 진행(런 밖 성장) 같은 구조 변경 제안은 이 문서의 범위 밖이다.
신선함 문제는 이 전제 안에서 해결한다.

## 3. 난이도 원칙

**방치(idle)는 죽고, 적극적인 조작·빌드 선택은 보상받는다.**

- 특정 wave 도달 같은 수치 벤치마크는 두지 않는다 (2026-06-12 결정).
- 판단 기준은 정성적이다: 가만히 서 있거나 아무 보상이나 집는 플레이가
  살아남으면 너무 쉬운 것이고, 숙련된 조작과 빌드 계획이 생존 시간을
  유의미하게 늘리지 못하면 잘못 어려운 것이다.

## 4. 알려진 문제의식 (이 목표가 나온 이유)

- 보상 픽 한도(`REWARD_PICK_LIMIT = 12`) 소진 후: wave 20 미만이면 레벨업이
  골드 보너스로 대체되고, wave 20+는 후기 강화 풀로 넘어간다. 한도 소진
  이후의 성장 경로가 얇아서 후반 레벨업의 의미가 약하다.
- 보상/융합/상점 루프가 초반과 후반이 동일해서, 시스템이 새로 열리는
  느낌이 없다.

## 5. 평가 기준 (game-critic용 체크리스트)

critic은 플레이로그·코드를 보고 각 항목을 PASS / FAIL / 판정불가로 판정하고,
FAIL에는 근거(로그 시점, 코드 위치)를 인용한다. fixer의 수정 제안은
FAIL 항목을 해소하는 방향이어야 하며, §2 전제를 벗어나면 안 된다.

| # | 질문 | FAIL 신호 예시 |
|---|------|----------------|
| G1 | wave 30+ 구간에서 빌드가 질적으로 변하는 이벤트(새 세트 효과 발동, 융합 변경, 시너지 개방)가 계속 나타나는가? | 후반 플레이로그에 수치 증가 외 변화 없음 |
| G2 | 둘 이상의 세트를 함께 키울 전략적 이유가 있는가? | 단일 타입 몰빵이 항상 최적해 |
| G3 | 후반 보상 선택지가 유의미한 트레이드오프인가? | 자명한 1픽이 반복됨, 나머지 카드가 들러리 |
| G4 | 방치 플레이는 실패하는가? | 조작 입력 없이(또는 무작위 픽으로) 후반까지 생존 |
| G5 | 조작·빌드 선택이 생존에 기여하는가? | 숙련 플레이와 방치 플레이의 결과 차이가 미미 |
| G6 | 픽 한도 소진 이후에도 성장 경로가 존재하는가? | 한도 소진 시점부터 빌드가 사실상 동결 |
| G7 | 제안/변경이 단일 룸 + 웨이브 루프 전제를 지키는가? | 맵 구조·메타 진행을 건드리는 제안 |
